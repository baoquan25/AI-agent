Execute code in a secure sandbox environment.

**When to use this tool:**
* Running Python scripts or code snippets
* Testing code behavior in isolation
* Executing data processing tasks

**Two execution modes:**
1. Inline: Provide `code` to run directly
2. Write & Run: Provide `code` + `file_path` to save first, then execute

**Parameters:**
* code: Python code to execute
* timeout: Maximum execution time (default: 30s)
* file_path: Optional path to save code before running
* working_dir: Optional directory to cd into before execution

**Examples:**
```python
# Run inline
code="print('hello')"

# Write file & run
code="print('hello')", file_path="/workspace/hello.py"
```
